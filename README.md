# 🛡️ Telegram-бот и Mini Apps для выдачи VPN-ключей из панели Remnawave.

![GhastVPN Banner](https://s1.radikal.cloud/2026/03/06/Thumbnail58b7cf4136e874e6.png)

## ⚡ Умный VPN-бот с интеграцией Remnawave и автоматическим управлением ключами. Включает в себя Telegram бота и Mini Apps приложение для управления ключами/оплатой/помощью и тд.

[VPN Сервис](https://t.me/GorynychVPNbot) — Это готовый к запуску VPN сервис, интегрированный с Remnawave. Имеет свой личный кабинет Mini Apps для управления ключами, пополнения баланса, просмотра истории и многое другое.

---

## 🚀 Основные возможности

### Telegram бот
- Написан на TypeScript, Bun и GrammyJS, Drizzle;
- Имеет 4 кастомных сообщения клиентам, в зависимости от условий;
- Кнопки: Личный кабинет, помощь, тех.поддержка и многие другие;
- Админские команды: Пополнить баланс, сбросить демо ( будут дополняться );
- Уведомления о остатках баланса и уведомления из Remnawave при взаимодействия с ключами ( опционально, настраеваемое );
- Реферальная система ( бонус рублями или процентами от пополнения ).

### 🔑 Личный кабинет
- Написан на Next.js, TypeScript, Drizzle;
- Выдача бонусных рублей на баланс новичкам ( кол-во дней настраивается, расчет автоматический );
- Получение ключей / редактирование / удаление ( нельзя удалять новые ключи в течение 24 часов );
- Система баланса как с дробями так и без ( 3.3 рубля 100 рублей ключ или 4 рубля 120 рублей за ключ );
- Ежедневное списание баланса;
- Пополнение баланса через Юкасса, ТБанк, Platega;
- Возможность редактирование название ключа для удобства;
- Кнопки помощи, видеопомощи, FAQ;
- Интегрированы приложения из Remna (оставили Happ но можно добавлять легко другие, просто сгенерировав в Remna).


### 🗂️ Развертывание
Развертывание занимает не более часа. Желательно на том же сервере что и Remna но можно ставить и отдельно, все общение происходит через API ключи. 

Все поднимается парой команд через **Docker compoose**. Инструкция прилагается. 

## ⚙️ Технологии

- ⚡ **TypeScript** + **Bun**
- 🔗 API-интеграция с **Remnawave**
- 🤖 Telegram Bot API
- 🐘 PostgreSQL + Docker Compose
